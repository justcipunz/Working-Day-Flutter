# Курсовая работа
## Трекер задач - часть сервиса Working Day
Корпоративное приложение для управления проектами и задачами с интеграцией REST API.
Работу выполнил: студент БПИ-2310 Кажкаримов Асхат

Код для трекера задач находится в папке /lib/tracker.

## Основные возможности
- Просмотр задач с группировкой по дедлайнам
- Канбан-доска для визуализации workflow
- Создание/редактирование задач и проектов
- Управление участниками проектов
- Отслеживание прогресса выполнения
- Интеграция с системой авторизации

## Технологии
- **Язык**: Dart (Flutter 3.1)
- **Архитектура**: MVVM
- **Сеть**: REST API
- **Стиль**: Material Design 3
